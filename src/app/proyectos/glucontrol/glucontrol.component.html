Glucontrol

Participe en un proyecto en grupo el cual consitia en desarrollar una pagina web para personas diabeticas esta página se desarrollo por la necesidad de que apuntar las diabetes sea accesible para todo el mundo, en este proyecto en grupo se dividio en backend y frontend yo junto a otro compañero nos encargamos del backend y el los demas de el fronted pero aunque el grupo estuviese dividido en dos ramas para hacer tareas nos apollabamos unos a otros si se necesitaba ayuda en backend los que estaban en frontend venian a ayudar y viceversa. El backend se desarrollo usando Entity framework y c#, ademas de disponer de una base de datos. El frontend se desarrollo usando Angular, HTML, CSS3 y typescript
