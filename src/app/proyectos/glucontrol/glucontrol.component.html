Glucontrol

Participé en un proyecto en grupo que consistía en desarrollar una página web para personas diabéticas. Esta página se desarrolló por la necesidad de que apuntar las diabetes sea accesible para todo el mundo. En este proyecto en grupo se dividió en backend y frontend. Yo, junto a otro compañero, nos encargamos del backend y los demás del frontend. Pero aunque el grupo estuviese dividido en dos ramas para hacer tareas, nos apoyábamos unos a otros si se necesitaba ayuda en backend, los que estaban en frontend venían a ayudar y viceversa. El backend se desarrolló usando Entity framework y C#, además de disponer de una base de datos. El frontend se desarrolló usando Angular, HTML, CSS3 y TypeScript.
