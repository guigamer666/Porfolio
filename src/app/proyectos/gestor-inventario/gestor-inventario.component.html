Gestor Inventario - e-commerce

Gestor Inventario es una aplicación diseñada para la gestión eficiente de inventarios en plataformas de comercio electrónico. El sistema está desarrollado en .NET MVC para el frontend y .NET 7.0 en el backend, proporcionando una solución robusta, escalable y segura.

NOVEDADES

Este proyecto a sido actualizado a la ultima versión de .NET a la 9.0, esta aplicacion ha pasado de ser un simple gestor de inventario a tomar forma de un e-comerce

Características del Proyecto

  • Gestión de Datos: Operaciones CRUD (Crear, Leer, Actualizar, Eliminar) para usuarios, proveedores, productos, pedidos y su historial.
  • Autenticación Robusta: Soporte para autenticación con generación de tokens, incluyendo:
    • Autenticación simétrica.
    • Autenticación asimétrica con clave pública y privada fija.
    • Autenticación asimétrica con clave pública y privada dinámica.
  • Generación de Informes: Exportación de reportes en formato PDF del historial de pedidos y productos.
  • Notificaciones por Correo Electrónico: Alerta automática cuando el stock de un producto está bajo.
  • Registro y Acceso de Usuarios: Registro de nuevos usuarios con confirmación por correo electrónico.
  • Panel de Administración: Herramienta para gestionar cuentas de usuario.
  • Sistema Basado en Roles: Control de acceso por niveles según permisos asignados.
  • Pasarela de Pago PayPal: Integración con PayPal para procesar pagos.
  • Restablecimiento de Contraseña: Opción para restablecer la contraseña mediante un correo con una clave temporal.
  • Flexibilidad en la Autenticación: Posibilidad de alternar entre los modos de autenticación modificando el código.
  • Gestión de Usuarios: Alta y baja de usuarios individuales o en masa por parte del administrador.
  • Docker: Configuración lista para desplegar en contenedores Docker.
  • Redis: Integración con Redis para mejorar el rendimiento.
  • Función de Reembolso: Nueva funcionalidad para procesar reembolsos de pedidos.
