Gestor de Tareas

Desarrollé un Gestor de Tareas que permite a los usuarios crear sus propias cuentas y mantener un registro de sus tareas pendientes de manera efectiva. Las tecnologías utilizadas incluyen: Angular, Bootstrap, HTML5, JavaScript, CSS3, Firebase.

Los resultados del proyecto han ayudado a los usuarios a mantenerse organizados y a no olvidar las tareas pendientes, proporcionando una plataforma para registrarlas y realizar un seguimiento eficiente. El proyecto se finalizó en agosto de 2023.

El backend esta echo usando firebase
